# Elasticsearch Cookbook

## Installation

Prerequisites:

- [Elasticsearch](https://www.elastic.co/downloads/elasticsearch)
- Ruby
- Bundler

```
bundle install
```

Run Elasticsearch (defaults to `localhost:9200`):

```
elasticsearch.bat
```

Run the examples:

```
ruby ./examples/hello.rb
```
