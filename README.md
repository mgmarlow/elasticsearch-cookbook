# Elasticsearch Cookbook

## Installation

Prerequisites:

- [Elasticsearch](https://www.elastic.co/downloads/elasticsearch)
- Ruby
- Bundler

```
bundle install
```

Run Elasticsearch (defaults to `localhost:9200`):

```
elasticsearch.bat
```

Run the console:

```
./bin/console
```

Now you can interact with the example applications:

```rb
Megacorp.seed_employees
```
